# yyyerin

yyyerin의 프로젝트 모음 페이지입니다.

🔗 [yyyerin.co.kr](https://yyyerin.co.kr)

## 소개

유용하고 재밌는 서비스를 만드는 개발자 yyyerin의 프로젝트들을 한 곳에서 확인할 수 있는 허브 사이트입니다.

## 프로젝트 목록

| 프로젝트 | 설명 |
|---|---|
| ✨ portfolio | 예린의 개발 포트폴리오 |
| 🍱 LunchPing | 내 주변 맛집을 빠르고 간편하게 찾아보세요 |
| 🎬 bomPick | OTT 콘텐츠 메타 추천 서비스 |
| 📜 AllCert | 자격증 정보를 한 곳에서 찾아보세요 |
| 🏛️ jiwonnow | 나에게 맞는 정부지원사업을 자동으로 추천받으세요 |
| ☕ lotto | 커피 찾기 게임 |

## 기술 스택

- **React 18** + **TypeScript**
- **Vite**
- **Tailwind CSS**

## 로컬 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```
